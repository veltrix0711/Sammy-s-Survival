# Contributing

## Branching
- `main`: protected, release-ready. PRs only.
- `dev`: integration branch.

## Conventional Commits
- `feat:`, `fix:`, `chore:`, `docs:`, `test:`, `refactor:`, `perf:`

## CI locally
- Bash: `bash scripts/validate_repo.sh || true`
- PowerShell: `pwsh scripts/validate_repo.ps1`
